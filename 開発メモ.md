# ポートフォリオ vol.1（最新版）— 開発メモ

このサイト（ポートフォリオ本体＋推しサイト「YUMIKI WORLD」）の制作記録です。
技術選定や設計の判断、実装で工夫した点をまとめています。

> ※「▼ 自分の言葉で」の箇所は、制作者本人が理解・判断を書き込む欄です。

---

## 1. このサイトについて（目的）

- ポートフォリオを全面リニューアルし、長期的に使える資産にする
- 大学で実際に作った成果物（グラフィック／CG・映像／画像処理／Web）を集約する
- 推しサイト「YUMIKI WORLD」を独立サイトとして併設し、Web制作実績として位置づける

## 3. 設計・デザインの方針

- 骨格はミニマル（モノクロ＋差し色1色・大きめの日本語タイポ・余白・Bentoグリッド）
- ヒーローと差し色で自分らしさ（色・キャラ）を効かせる折衷
- 作品は画像の羅列でなく「課題 → 工夫 → 結果」のケーススタディ型で見せる
- 動きは「意味のあるモーション」（スクロール連動・控えめなホバー）

## 4. リニューアルで設計し直したこと

- 全ページにコピペされていたヘッダー／フッター／メニュー → 共通コンポーネント化
- リンク切れ・空ファイル・意味不明な画像名 → 構成を整理し、命名を統一
- 欠けていたメタ情報（description／OGP／favicon）を整備
- ページ遷移の体感速度を改善
